# Lemon Squeezy Setup Guide

**Purpose:** Step-by-step instructions to create lamco-rdp-server products
**Store URL:** https://lamco.lemonsqueezy.com

---

## Before You Start

1. Log in to https://app.lemonsqueezy.com
2. Ensure your store is set up (name, logo, support email)
3. Have your payout method configured

## After Creating Products

### Get Checkout URLs

For each product:
1. Go to **Products**
2. Click on the product
3. Click **Share** or look for the checkout URL
4. Copy the URL (format: `https://lamco.lemonsqueezy.com/buy/XXXXXXXX`)

Record them here:

| Product | Checkout URL |

## Website Integration

### Button/Link Format

```html
<!-- Simple link -->
<a href="https://lamco.lemonsqueezy.com/buy/XXXXXXXX">Buy License</a>

<!-- Styled button -->
<a href="https://lamco.lemonsqueezy.com/buy/XXXXXXXX"
   class="btn btn-primary">
  Buy Monthly License - $4.99/mo
</a>
```

### Checkout Overlay (Optional)

Lemon Squeezy provides a JavaScript library for overlay checkout:

```html
<script src="https://assets.lemonsqueezy.com/lemon.js" defer></script>

<a href="https://lamco.lemonsqueezy.com/buy/XXXXXXXX"
   class="lemonsqueezy-button">
  Buy License
</a>
```

This opens checkout in an overlay instead of navigating away.
